## ClassRegistrationProject

<프로젝트 개요><br>

본 프로젝트는 “한성대학교 종합정보시스템” 기능 중 몇 가지 기능을 Spring Framework를 통해 구현한다.<br>

1. Spring MVC <br>
2. Spring JDBC<br>
3. Spring Security 5 <br>
4. Sping Web Form <br>
*Data Binding, 
*Data Validation, 
*Data Buffering<br><br>
5. loging

위의 기능을 통해서 수강신청 내역에 관하여 CRUD를 기능을 Spring MVC 패턴에 맞추어 구현하였다.
